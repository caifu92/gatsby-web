## Installation

### Step 1: Run the following command to create a new site

```shell
gatsby new site arshad/gatsby-starter-flex
```

### Step 2: Configure `.env`

Copy `.env.example` to `.env` and update `SITE_URL`.

### Step 3: Start developing

```shell
cd my-site/
gatsby develop
```
